## Extensions

- **Local LLMs with lab GPU** — run larger open models locally against the sandbox and compare
  behavior to the built-in explainer.
- **API-based model comparison (synthetic data only)** — compare a cloud model to the local
  one **using only synthetic logs / mock telemetry**; never send real or subscriber data.
- **O-RAN / xApp / rApp boundary extensions** — map the sandbox onto near-RT / non-RT RIC
  roles and their trust boundaries.
- **ISAC and robot-car simulation extensions** — richer sensing-confusion and cyber-physical
  consequence scenarios.
- **Research project ideas** — new attack classes, new safety gates, new evidence/provenance
  types, or transfer to captured-but-labelled data from *induced* failures on a testbed you
  control.

> Any extension inherits the [Sandbox Safety Boundary](../../README.md#sandbox-safety-boundary):
> synthetic/mock data, no real actuation, evidence-based reasoning.
